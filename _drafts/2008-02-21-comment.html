def. 1 - A sentence S1 implies sentence S2 iff its schema M1 implies schema M2.
def. 2 - A schema M1 implies schema M2 iff every time M1 is true, M2 is true also.

Equivalence
def. 1 - Two sentences are equivalent iff their schemas are equivalent.
def. 2 - Two schemas are equivalent iff whatever assignments of truth values to one schema that gives the schema a certain truth value (T or F) gives the other schema the same truth value.
--> Two schemas are equivalent iff they have the same truth table.
Theorem - Two schemas are equivalent iff they imply one another (both directions of implication are met, so to speak).
--> All tautologies are equivalent to each other; all contradictions are equivalent to each other.

Theorem - Schema M1 is equivalent to M2 iff M1≡M2 is a tautology.
Schema M1 implies Schema M2 iff M1⊃M2 is a tautology.

Consistency
def. 1 - A sentence is consistent iff its schema is consistent.
def. 2 - A schema is consistent iff it comes out true for at least one assignment of truth values to letters p, q, etc.
